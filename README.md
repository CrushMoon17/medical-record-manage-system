1、public目录： 项目公共目录，存放静态资源(img、js、css)和公共资源，404错误提示页面；

3、views目录：  视图/模版目录，存放HTML模板文件；

4、module目录： 自己写的一些模块；

5、node_modules：存放所有的项目依赖库，配置package.json之后执行npm install后自动创建的文件夹；

6、app.js： 项目入口文件，程序初始运行文件；

7、package.json ：项目配置信息，依赖模块的定义
